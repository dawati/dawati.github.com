Dawati

Dawati is an open source project to make computers that help people get things done. Dawati has been used by the Moblin, MeeGo and Tizen projects to deliver a user interface to millions of people worldwide. The main target for Dawati is Netbooks with Intel Atom processors, 3D graphics, a small (10") screen, a keyboard and a bad trackpad (although good trackpads make us happy). Over time we'd like to expand our target to multiple monitors, touch screens, and more powerful processors. You can help us do this.

Get Dawati

    • Developing

    Dawati is made up of lots of different projects that you need to glue together to make a usable system:

    • dawati-shell - the toolbar and default set of panels to handle a minimally useful experience
    • dawati-artwork - non dawati specifc theming for sound, gtk+, pointers etc. helps create a unified system feel
    • dawati-help - will link to documentation
    • bisho - an application to log into social networks
    • dawati-jhbuild - helps developers build some or all of dawati on their systems

    Dawati works with and relies upon the invaluable contribution from a number of other free software and open source communities, in particular: GNOME, Clutter, GTK+, Banshee and Media Explorer. Not everything you need to make a Dawati based system is hosted here.
